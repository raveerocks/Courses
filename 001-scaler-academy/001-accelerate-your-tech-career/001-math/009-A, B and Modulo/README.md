# A, B and Modulo

## Problem Description

Given two integers **A** and **B**, find the greatest possible positive **M**, such that the following condition is satisfied: **A%M = B%M** Return **M**. 

## Problem Constraints
```
1 <= A, B <= 10^9
A != B
```
