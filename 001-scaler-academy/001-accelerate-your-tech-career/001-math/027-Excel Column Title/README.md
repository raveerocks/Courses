# Excel Column Title

## Problem Description
Given a positive integer A, return its corresponding column title as appear in an Excel sheet. 

For example:
```
    1 -> A
    2 -> B
    3 -> C
    ...
    26 -> Z
    27 -> AA
    28 -> AB 
```
## Problem Constraints
```
1 <= A <= 10^9
```

## Input Format
```
First and only argument of input contains single integer A
```

## Output Format
```

```
## Example
```
Input :
  A = 27

Output 1:
 "AA"
```

## Example Explanation
```
    1 -> A,
    2 -> B,
    3 -> C,
    ...
    26 -> Z,
    27 -> AA,
    28 -> AB 
```
